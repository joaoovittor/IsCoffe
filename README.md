# IS Coffee 

A modern front-end study project focused on building a coffee shop landing page with interactive JavaScript features and a clean, Pinterest-inspired design.

## About

IS Coffee is a fictional coffee shop website created to practice HTML, CSS, and vanilla JavaScript.

The project simulates a modern café, featuring handcrafted drinks, an interactive menu, image gallery, smooth scrolling navigation, scroll animations, and an accordion to provide a realistic user experience.

- Smooth scrolling navigation

- Modern Pinterest-inspired UI

## Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)

## About the Project

This project was built as part of my front-end learning journey after completing the introductory JavaScript DOM module. It focuses on applying DOM manipulation concepts by creating a modern coffee shop website with interactive user interface components.

The main goal was to reinforce concepts such as smooth scrolling, tab navigation, scroll animations, and accordion components while improving code organization and front-end development skills.
